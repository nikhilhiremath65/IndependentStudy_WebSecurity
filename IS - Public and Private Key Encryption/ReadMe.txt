The 'IS - Public and Private Key Encryption' directory has three components:

1) Public and Private Key Encryption.ppt
	- Class Sides for Public and Private Key Encryption

2) Diagrams Public and Private Key Encryption.draw.io
	- A draw.io file that contains all the diagrams used in the slides.
	
3) RSA-Assignment folder
	- Assignment to get familiarized with Public Key Encryption by 		            implementing a RSA algorithm.
	- RSA Assignment.doc/.pdf(Assignment Instructions)
	- RSA_Given (Intial code - to be give to students)
	- RSA2_Solution (Complete Solution)
